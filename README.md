# Bellabeat Case Study  
### Smart Device Usage Analysis & Marketing Recommendations  

This project analyzes Fitbit smart device usage data to uncover trends in user activity, sleep, and calorie expenditure.  
Based on these insights, I developed strategic marketing recommendations for Bellabeat, a wellness technology company focused on women's health.
